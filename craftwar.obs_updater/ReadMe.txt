This is craftwar's customized OBS Studio. (improve performance, add/remove features ex: display playing song)
If You need more features, you need install yourself or use official OBS Studio version.

"update updater.cmd" to update updater
"update OBS.cmd" to update OBS
"update and run OBS.cmd" to update and start OBS
"no gc.cmd" update OBS except game capture
"extract OBS.cmd" extract OBS (no download)

This updates craftwar's OBS studio only. (not including plugins you install)


If you can't run OBS, you should download Microsoft Visual C++ Redistributable for Visual Studio 2017 (x64) first.
If 2015 Redistributable is installed, you can run program. (may work abnormally) Install 2017 please. (no uninstall 2015/2017 is needed)
https://support.microsoft.com/help/2977003/the-latest-supported-visual-c-downloads

If you need browser plugin, you should download from
https://github.com/obsproject/obs-browser/releases
(no auto update to this plugin)
